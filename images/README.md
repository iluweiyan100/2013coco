# 📸 首页图片资源清单

## 从 Figma 导出的图片

根据你的 Figma 设计稿，需要导出以下图片：

### 1. Banner 轮播图

**文件**: `images/banner.jpg`  
**尺寸**: 750×480 px (或更大)  
**格式**: JPG 或 PNG  
**说明**: 首页顶部的宣传图片

**导出步骤**:
1. 在 Figma 中找到 Banner Frame（节点 ID: 1:10）
2. 右侧面板 → Export
3. 选择 JPG 格式，Scale: 2x
4. 保存到 `images/` 目录
