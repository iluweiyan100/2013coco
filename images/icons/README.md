## 🔧 如果需要更精确的图标

### 方法 1：从 Figma 直接导出 SVG（推荐）

1. **在 Figma 中选中图标组**
   - 堂食图标：选中节点 `1:19` (Icon Frame)
   - 外带图标：选中节点 `1:28` (Icon Frame)

2. **导出步骤**
   - 右侧面板 → Export
   - 选择 **SVG** 格式
   - Scale: 1x
   - 点击 Export

3. **保存文件**
   - 堂食图标保存为：`images/icons/dine-in.svg`
   - 外带图标保存为：`images/icons/takeaway.svg`

4. **替换文件**
   - 将导出的 SVG 文件覆盖到对应位置即可
