# 🎨 点单页面 - 商品卡片设计规范

## 📐 Figma 设计规格

### 卡片容器
- **尺寸**: 265×104px (530×208rpx)
- **背景色**: #FFFFFF (纯白)
- **圆角**: 8px (16rpx)
- **阴影**: `0 2px 8px rgba(0, 0, 0, 0.06)`
- **内边距**: 24rpx (12px)
- **间距**: 卡片之间 margin-bottom: 24rpx

### 布局结构
```
┌─────────────────────────────────────┐
│  [图片]  [名称]              [+]   │
│  160×160  [价格]                    │
│          [描述]                     │
└─────────────────────────────────────┘
```

## 📝 文本信息区

### 商品名称
- **字号**: 16px (32rpx)
- **字重**: 500 (Medium)
- **颜色**: #333333
- **行高**: 1.4
- **溢出处理**: 单行截断 + 省略号

### 商品价格
- **字号**: 16px (32rpx)
- **字重**: 600 (SemiBold)
- **颜色**: #A3212D (品牌红)
- **行高**: 1.2
- **格式**: ¥XX (人民币符号 + 整数)

### 商品描述
- **字号**: 12px (24rpx)
- **颜色**: #999999
- **行高**: 1.3
- **溢出处理**: 单行截断 + 省略号
- **显示条件**: 仅当 description 存在时显示

## ➕ 添加按钮

### 按钮容器
- **尺寸**: 32×32px (64×64rpx)
- **形状**: 圆形 (border-radius: 50%)
- **背景色**: #A3212D (品牌红)
- **左边距**: 8px (16rpx)
- **交互反馈**: 
  - 点击时透明度 0.7
  - hover-stay-time: 100ms

### 加号图标
- **尺寸**: 20×20px (40×40rpx)
- **线条粗细**: 2px (4rpx)
- **颜色**: #FFFFFF (纯白)
- **实现方式**: CSS 伪元素绘制十字

```css
.plus-h { /* 横线 */
  width: 40rpx;
  height: 4rpx;
}
.plus-v { /* 竖线 */
  width: 4rpx;
  height: 40rpx;
}
```

## 💡 实现要点

### 1. Flexbox 布局
```css
.product-card {
  display: flex;
  align-items: center; /* 垂直居中 */
}

.product-info {
  flex: 1; /* 占据剩余空间 */
  min-width: 0; /* 允许文本截断 */
}
```

### 2. 文本溢出处理
```css
.product-name,
.product-desc {
  overflow: hidden;
  text-overflow: ellipsis;
  white-space: nowrap;
}
```

### 3. 图片懒加载
```html
<image src="{{item.image}}" lazy-load></image>
```

### 4. 按钮点击反馈
```html
<view hover-class="btn-active" hover-stay-time="100">
```

## 🚀 性能优化建议

1. **图片优化**
   - 使用 WebP 格式（如果支持）
   - 压缩图片质量至 75-85%
   - 启用懒加载 (`lazy-load`)

2. **列表渲染**
   - 使用 `wx:key` 提高渲染性能
   - 考虑虚拟列表（商品数量 > 50 时）

3. **样式优化**
   - 避免使用复杂的选择器
   - 减少嵌套层级
   - 使用 CSS 变量管理颜色

## 📱 响应式适配

当前设计基于 iPhone 12/13/14 (390px 宽度)：
- 左侧分类栏: 192rpx (96px)
- 右侧商品区: 自适应宽度
- 商品卡片: 530rpx (265px)

在不同设备上会自动适配，因为使用了 rpx 单位。
